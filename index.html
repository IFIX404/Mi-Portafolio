¡Hola! Soy Wiliam Patiño

Un apasionado del desarrollo web y de la tecnología. Con una formación en electricidad industrial y habilidades en programación.

Comencé mi viaje en el desarrollo durante mis estudios y desde entonces he estado aprendiendo y explorando nuevas tecnologías. Me encanta trabajar con HTML, CSS y JavaScript para crear interfaces dinámicas y atractivas. Utilizo herramientas como Sass y Bootstrap para optimizar mi flujo de trabajo y acelerar el proceso de diseño.
También tengo experiencia en el desarrollo del servidor utilizando Spring Boot, un framework de Java para crear aplicaciones web escalables. Adopto prácticas de desarrollo colaborativas utilizando Git y GitHub. Me he capacitado en programas de Alura Latam y Oracle para ampliar mis conocimientos técnicos.
Soy una persona curiosa y autodidacta, siempre buscando oportunidades para aprender y crecer profesionalmente. Estoy emocionado de ser parte de este apasionante mundo de la tecnología y estoy disponible para colaborar en proyectos interesantes. ¡No dudes en contactarme si tienes alguna pregunta o propuesta!
